# FABulous BRAM Instatiation

An example VGA design for FABulous FPGAs demonstrating how BRAMs and other primitives can be instantiated in FABulous.

The example is taken from [here])(https://github.com/gatecat/fabulous-mpw2-bringup/tree/main/sim/test_design).

To build the design, go into `Test` and run
```shell
make build_test_design
```

> [!NOTE] 
> The testbench is still the default testbench from fabulous and does not actually test the design, but you can still use the Makefile to build the design.
